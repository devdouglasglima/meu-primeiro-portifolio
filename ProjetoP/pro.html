calculadora


Este projeto tem como objetivo desenvolver uma calculadora de equações do segundo grau, também conhecidas como equações quadráticas. Uma equação do segundo grau possui a forma geral ax² + bx + c = 0, onde a, b e c são coeficientes reais fornecidos pelo usuário. A calculadora permitirá que o usuário insira os valores de a, b e c, e então calculará as raízes da equação utilizando a fórmula de Bhaskara:
